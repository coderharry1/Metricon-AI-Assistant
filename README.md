# Metricon HomeIQ â€” AI Assistant 

Metricon HomeIQ is a POC that explores what it looks like when a home builder uses GenAI to actually answer customer questions, not just point them to a brochure.

It's built on a RAG pipeline using AWS Bedrock Nova and Qdrant, pulling answers directly from Metricon's own documents. Ask it anything about costs, the building process, or first home buyer options, and it tells you exactly where the answer came from.

The idea is simple: if this works for customers, the same approach could help Metricon's internal teams, too. 

Construction, planning, approvals, compliance, any team drowning in documents could use something like this.

Most RAG pipelines just split text and store it. **ConstructIQ goes further.**

Every single chunk is sent to **AWS Bedrock Nova**, which reads it, understands it, and enriches it with:

| Field | Description |
|-------|-------------|
| `title` | Short descriptive title generated by AI |
| `summary` | 50-word AI-generated summary |
| `keywords` | 5 extracted keywords for better search |
| `category` | Auto-classified topic area |
| `importance` | AI-scored relevance from 1â€“10 |

This means when a customer asks a question, we're not just doing keyword matching â€” we're matching against **AI-understood meaning**.

## âš¡ Quick Start

### 1. Clone & Setup
```bash
git clone https://github.com/YOUR_USERNAME/metricon-constructiq.git
cd metricon-constructiq
python -m venv venv && source venv/bin/activate
pip install boto3 python-dotenv qdrant-client sentence-transformers gradio pypdf
```

### 2. Configure AWS
Create `.env`:
```env
AWS_ACCESS_KEY=your_access_key_here
AWS_SECRET_ACCESS_KEY=your_secret_key_here
```

### 3. Start Qdrant
```bash
docker run -p 6333:6333 -v $(pwd)/qdrant_data:/qdrant/storage qdrant/qdrant
```

### 4. Run the Pipeline
```bash
# Step 1: Chunk and enrich PDFs with AI
python chunker/chunker.py

# Step 2: Embed and ingest into Qdrant
python ingestion/data_ingestion.py

# Step 3: Launch the chatbot
python app.py
```

### 5. Open in browser
```
http://localhost:7860
```
